# Chronic Kidney Disease Analysis Using SVM and Decision Tree

## Project Overview
Chronic Kidney Disease (CKD) is a serious health condition that affects kidney function and can lead to severe complications if not detected early. This project focuses on the analysis and prediction of CKD using machine learning techniques. A comparative study was performed using Support Vector Machine (SVM) and Decision Tree classifiers to evaluate their effectiveness in disease prediction.

## Objectives
- Analyze Chronic Kidney Disease data using machine learning techniques.
- Develop predictive models for CKD classification.
- Compare the performance of SVM and Decision Tree algorithms.
- Identify the most effective model for disease prediction.

## Results
The performance of SVM and Decision Tree classifiers was evaluated using the CKD dataset. The Decision Tree classifier demonstrated better overall performance and interpretability compared to the SVM model for the selected dataset.
